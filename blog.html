The Difference Between 'While' and 'Do While' loops

The difference between 'while' and 'do while' loops is that in a 'do while' loop the code always executes once before the condition is checked. In a while loop, the condition is checked before the code executes.

java.util.Scanner;

When you import java.util.Scanner; into your code you are importing a library and function that allows a user to enter input into the program.

'void' method declaration

The 'void' method declaration is for functions that do not return a data type. If a function has a value to return then it must be declared. If it does not, then it must be declared void.
